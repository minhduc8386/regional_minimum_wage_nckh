# Province Wage Region Mapping Notes

- This mapping is a province-level approximation used to merge regional minimum wages into the province-year panel.
- Vietnam's regional minimum wage is officially applied at district/urban district/town/provincial-city areas, not always at the whole-province level.
- Because the current research dataset is province-year aggregate data, this mapping assigns one wage region to each province-year.
- If a province contains districts in multiple wage regions, the assigned wage region is an approximation and should be treated as a data limitation.

- Rows flagged as mixed district regions: 385.
